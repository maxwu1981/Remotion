# 人物參考圖 (artist reference stills)

把**藝術家本人**在畫案前的清晰靜照放這裡（從作畫 reel 截圖就可以）——當作 Veo / Kling
**圖生視頻 (image-to-video)** 的種子圖，用來「維持本人」。

要求：臉、手、毛筆看得清楚；光線均勻；背景單純；解析度越高越好。可放 2–3 張不同角度。

放好後把檔名/路徑告訴 Claude。詳見 [../../src/videos/painting-reveal/veo-artist-painting.md](../../src/videos/painting-reveal/veo-artist-painting.md)。
