Claude Code 的記憶與 CLAUDE.md 怎麼運作、會遇到什麼坑？

Claude Code 透過 CLAUDE.md 與記憶機制保留專案脈絡。頻道有實戰影片說明記憶、CLAUDE.md、容器、沙盒、遠端與手機操作實際會遇到的坑與對應解法。

MCP 和 API 差在哪、何時該用哪個？

MCP（Model Context Protocol）像 AI 的「萬能轉接頭」，把工具串接標準化；API 則是直接呼叫服務。頻道有一支「MCP vs API」實戰影片解析兩者差異與選擇時機。

如何用 Remotion 搭配 Claude 打造自動化影片產線？

用程式碼（Remotion）結合 Claude 自動剪輯、配音與算圖，組成每日影片產線。頻道有完整的踩坑紀錄與實作教學。
